RVone

RVone[ghtt_,ghbb_,ghZZ_,x_,xmin_,xmax_,xlabel_][[i]]
This command evaluates

with V=W, Z when there is dependence only on one parameter. The arguments ghtt, ghbb, ghVV are the htt, hbb, hVV couplings. Here, h represents to SM-like Higgs boson while t and b are the top and bottom quarks. The label x indicates the parameter to constraint, while xmin and xmax are the initial and final values defined by users and xlabel is used for indicates the X axis label. Finally, [[i]] stands for confidence level, i=1 (2) indicates 1σ (2σ).
