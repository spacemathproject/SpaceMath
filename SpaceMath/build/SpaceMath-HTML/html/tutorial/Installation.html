SPACE MATH TUTORIAL

Installation

SpaceMath can be installed on any computer that runs Mathematica 8.0 or higher. However, future releases of SpaceMath will require Mathematica 10.1 or higher.

Automatic Installation

Run the following instructions in a Mathematica session: Span[{class -> TI}, XMLElement[a, {href -> http://reference.wolfram.com/language/ref/Import.html}, {Import}], [, h, XMLElement[span, {class -> TI}, {ttps}], XMLElement[span, {class -> TI}, {:}], XMLElement[span, {class -> TI}, {//}], XMLElement[span, {class -> TI}, {raw}], ., githubusercontent, ., com, /, spacemathproject, /, SpaceMath, /, master, /, SpaceMath, /, Install, ., m, "]]
InstallSpaceMath[]

Manual installation

The latest version of the program, including expansion packs, is hosted at the page Span[{class -> TI}, https, :, //, github, ., com, /, spacemathproject, /, XMLElement[a, {href -> ../ref/SpaceMath.html}, {SpaceMath}], /, archive, /, master, ., zip]

After downloading and uncompressing the zip file, place the folder SpaceMath and all its contents within the directory $UserBaseDirectory/Applications/. Here, $UserBaseDirectory can be found by running $UserBaseDirectory within a Mathematica session:

Out[1]=

This should install SpaceMath on your system.
